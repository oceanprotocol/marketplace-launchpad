# NFTs & Smart Contracts

## Outcome

Understanding NFT & V4 Smart Contract functionalities; ERC20 Enterprise Token template (v4).

## Description

This interactive opening module provides the foundation for individuals and businesses to think of data more like an asset rather than information. Starting with a high level overview of Ocean Protocol this module will examine different components of the Ocean architecture and explain how Ocean enables data sharing while maintaining privacy. By the end of the session participants will know how to publish and consume data on Ocean Protocol.
