# Support

## Community Support

Please don't hesitate to reach out if you have any questions - we're here to help!

### Discord

- Ask us questions anytime via [discord](https://discord.gg/TnXjkR5)
- Use the **dev-support** channel to ask any technical questions that you have
- Questions will be answered by core team and community members
- Our Discord is a thriving community and a great place to ask technical or non-technical questions
- We aim to give responses within 24 hours

### GitHub

- If you've found an issue with our code or any of the lesson content here, then please [open an issue on GitHub](https://github.com/oceanprotocol/marketplace-launchpad/issues/new/choose)
- We also welcome contributions from community members, so please don't hesitate to open a pull request.

### Telegram

- Join the latest discussions on Ocean Protocol on our [Telegram](https://t.me/oceanprotocol_community)
- Telegram is a great place for general discussions and non-technical questions.
